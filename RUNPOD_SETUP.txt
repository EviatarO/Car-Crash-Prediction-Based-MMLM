==============================================================
  RunPod Setup — Car Crash Prediction MMLM
  Run these commands every time you spin up a NEW pod.
  All code fixes are in git — just pull and install.
==============================================================

--------------------------------------------------------------
STEP 1 — Clone or pull the repo
--------------------------------------------------------------
# If fresh pod (no repo yet):
cd /workspace
git clone https://github.com/EviatarO/Car-Crash-Prediction-Based-MMLM.git
cd Car-Crash-Prediction-Based-MMLM

# If repo already exists:
cd /workspace/Car-Crash-Prediction-Based-MMLM
git pull

--------------------------------------------------------------
STEP 2 — Install dependencies
--------------------------------------------------------------
pip install -r requirements.txt
pip install flash-attn --no-build-isolation

--------------------------------------------------------------
STEP 3 — Keep HuggingFace cache on LOCAL disk (NOT /workspace)
         /workspace has ~10 GB quota — model is 7.4 GB, leave it on root.
         Local disk has ~20 GB free and is fine for the model cache.
--------------------------------------------------------------
# 3a. Remove any stale /workspace/.cache from previous sessions
#     (this filled the 10 GB quota in past sessions — always clean it):
rm -rf /workspace/.cache

# 3b. Fix /root/.cache/huggingface ONLY if it is a broken symlink.
#     If it is already a real directory with the model inside, SKIP this
#     (running rm would delete 7.4 GB and force a re-download).
ls -la /root/.cache/huggingface
# If output shows: lrwx... -> /workspace/...  (a symlink) → run the fix:
#   rm -rf /root/.cache/huggingface && mkdir -p /root/.cache/huggingface
# If output shows: drwx...               (a real directory) → nothing to do.

# /workspace disk usage should be:
#   ~1.1 GB  data
#   ~1.2 GB  repo
#   ~0.5 GB  checkpoints (grows during training)
#   Total: well under 10 GB quota

--------------------------------------------------------------
STEP 4 — Start training
--------------------------------------------------------------
cd /workspace/Car-Crash-Prediction-Based-MMLM

PYTORCH_CUDA_ALLOC_CONF=expandable_segments:True nohup python student_training/scripts/train_lora.py \
    --jsonl outputs/teacher_dataset_v11.jsonl \
    --frames_root /workspace/data/extracted_clips/train \
    --config student_training/configs/train_lora.yaml \
    > train_run.log 2>&1 &

tail -f train_run.log

--------------------------------------------------------------
STEP 5 — Resume if training crashed (picks up last checkpoint)
--------------------------------------------------------------
PYTORCH_CUDA_ALLOC_CONF=expandable_segments:True nohup python student_training/scripts/train_lora.py \
    --jsonl outputs/teacher_dataset_v11.jsonl \
    --frames_root /workspace/data/extracted_clips/train \
    --config student_training/configs/train_lora.yaml \
    --resume \
    > train_run.log 2>&1 &

tail -f train_run.log

--------------------------------------------------------------
STEP 6 — Monitor
--------------------------------------------------------------
# Check process is alive:
ps aux | grep train_lora

# Check GPU usage (expect ~12-14 GB per card on 2x RTX 4090):
nvidia-smi

# Check disk space:
df -h
du -sh /workspace/* /workspace/.cache 2>/dev/null | sort -rh

# See latest log:
tail -50 train_run.log

NOTES
--------------------------------------------------------------
- Recommended GPU: 2x RTX 4090 (~$1.18/hr) — 48 GB combined, model splits automatically
- /workspace quota is ~10 GB — DO NOT put HF model cache there
- HF model cache (~7.4 GB) stays on local root disk (/root/.cache/huggingface)
- Training: 50 epochs x ~5 min/epoch = ~4 hrs = ~$5 total
- Checkpoints saved every 5 epochs to: outputs/checkpoints/e2_lora_100clips/
- All 50 checkpoints kept (save_total_limit=50, ~100 MB each = ~5 GB total in /workspace)
- After training: pick best epoch from epoch_metrics.jsonl (highest val_f1, smallest train-val gap)
- Data paths on RunPod:
    JSONL  : /workspace/Car-Crash-Prediction-Based-MMLM/outputs/teacher_dataset_v11.jsonl
    Frames : /workspace/data/extracted_clips/train/{video_id}/frame_00042.jpg
