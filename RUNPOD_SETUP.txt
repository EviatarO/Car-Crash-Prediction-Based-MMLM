--------------------------------------------------------------
STEP 0 — Select the right RunPod template (RTX 5090 ONLY)
--------------------------------------------------------------
  RTX 5090 (Blackwell) requires PyTorch 2.8+ and CUDA 12.8+.
  When creating the pod, choose a template such as:
    "RunPod PyTorch 2.8.0"  (or any image with PyTorch >= 2.8 + CUDA 12.8)
  DO NOT use a PyTorch 2.1/2.2/2.4 image — the GPU will not be recognised.

  Attach a Network Volume (at least 30 GB) to /workspace.
  This is where frames, checkpoints, and outputs are stored.

  Verify GPU after pod starts:
    nvidia-smi
  Expect: RTX 5090, 32 GB VRAM.

  Verify PyTorch sees the GPU:
    python -c "import torch; print(torch.__version__, torch.cuda.get_device_name(0))"

  If flash-attn errors after pip install, rebuild for your CUDA version:
    pip install flash-attn --no-build-isolation --force-reinstall

STEP 4 — Upload hires frames  [E3a — do ONCE per workspace volume, from LOCAL machine]
--------------------------------------------------------------
# The old 256-px frames ({vid}/) are already on the network volume from E2.
# For E3a we only need to upload the NEW _hires dirs (107 folders x 16 frames).

# --- 4a. Zip only the _hires dirs (run on LOCAL Windows PowerShell / terminal) ---

cd "C:\Users\eviatar.ohayon\Ramon Space\PycharmProjects\Thesis\MMLM_For_Cars_Collision_Anticipation\MMLM_AI"

python -c "
import zipfile, glob, os
from pathlib import Path
src  = Path('dataset/train')
out  = Path('outputs/extracted_clips_hires.zip')
dirs = sorted(src.glob('*_hires'))
print(f'Zipping {len(dirs)} hires dirs...')
with zipfile.ZipFile(out, 'w', zipfile.ZIP_DEFLATED) as zf:
    for d in dirs:
        for f in sorted(d.glob('frame_*.jpg')):
            arcname = 'train/' + d.name + '/' + f.name
            zf.write(f, arcname)
sz = out.stat().st_size / 1024**2
print(f'Done: {out}  ({sz:.0f} MB)')
"
# Expected output: Zipping 107 hires dirs...  Done: outputs/extracted_clips_hires.zip  (~300-500 MB)

NOTES
--------------------------------------------------------------
- Recommended GPU  : 1x RTX 5090 (~$0.80-1.00/hr) — 32 GB VRAM, fits model with flash-attn + grad-ckpt + bf16
- Fallback GPU     : 2x RTX 4090 (~$1.18/hr) — multi-GPU patches fire automatically when n_gpus > 1
- /workspace quota : watch disk space (use STEP 9 df -h); keep HF model cache on root disk, NOT /workspace
- HF model cache   : ~7.4 GB stays on /root/.cache/huggingface (local disk, NOT /workspace)
- Training time    : 89 clips x 50 epochs x ~2s/clip ~= 2.5 hrs on RTX 5090  (~$2-3 total)
- Checkpoints      : saved every 50 optimizer steps to outputs/checkpoints/e3a_lora_89clips/
                     (~100 MB each x ~60 checkpoints = ~6 GB total in /workspace)
- Best checkpoint  : pick from epoch_metrics.jsonl — highest val_ap (primary metric) or val_f1
- Val set          : 18 GT clips (9 YES / 9 NO) — HUMAN ground truth, held out from training
- Train set        : 89 teacher-distilled clips (42 YES / 47 NO) — hires 1280x720 frames
- Frame dirs       : {video_id}_hires/ (16 frames numbered frame_00001..frame_00016)
- Data paths on RunPod:
    Train JSONL : /workspace/Car-Crash-Prediction-Based-MMLM/outputs/teacher_dataset_e3a.jsonl
